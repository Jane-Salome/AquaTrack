# AquaTrack 🌊

**Live Demo** → [Click here to view the app](https://enchanting-jelly-5ebe2b.netlify.app/)

A crowd-sourced ocean hazard reporting and monitoring platform that aggregates citizen reports and social media data to track real-time coastal hazards across India — improving public safety and disaster awareness.

---

## Features

- **Interactive Map** — Leaflet.js powered map showing live hazard locations across Indian coastal regions
- **Crowd-sourced Reports** — Aggregates hazard data from citizen reports and social media platforms (Twitter, Facebook, Instagram, YouTube)
- **Real-time Filtering** — Filter hazards by event type, data source, and date
- **Live Stats Dashboard** — Displays total reports, active hazards, and social media mention counts
- **Event Categories** — Tracks Tsunamis, Storm Surges, and High Wave alerts
- **Social Media Feed** — Live feed of hazard reports from multiple platforms

---

## How It Works

1. **Data Aggregation** — Collects hazard reports from citizens and social media platforms
2. **Map Visualization** — Plots each hazard as a marker on an interactive Leaflet map centered on India
3. **Filtering** — Users filter by event type (Tsunami, Storm Surge, High Waves), source (Citizen/Social Media), or date
4. **Live Stats** — Dashboard dynamically updates total reports, active hazard count, and social media mentions based on active filters
5. **Feed Display** — Social media feed shows real-time hazard alerts with source attribution

---

## Monitored Regions

- Visakhapatnam (Vizag) — High Waves
- Kerala Coast — Storm Surge
- Chennai — Tsunami Alerts
- Kanyakumari — Ocean Swells
- Goa — Local Coastal Alerts

---

## Use Case

India's coastline spans over 7,500 km and is highly vulnerable to ocean hazards. AquaTrack bridges the gap between official monitoring systems and ground-level reporting — empowering communities to report and track hazards in real time, reducing response time and improving coastal safety.
